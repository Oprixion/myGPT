We are currently working on building a GPT following the "Attention is All You Need" paper, going from tokenization and a bigram baseline all the way through self-attention, multi-head attention, feedforward layers, residual connections, and layer normalization. The project will closely follow the tutorial of Andrej Karpathy's Let's build GPT (https://www.youtube.com/watch?v=kCc8FmEb1nY). 

The github repo for which we aim to replicate is: https://github.com/karpathy/nanoGPT/tree/master
